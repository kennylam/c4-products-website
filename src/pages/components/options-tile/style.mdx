## Structure

| Size variant | Element | Property | px / rem | Spacing token |

| | Heading | `padding-right` | - | `$spacing-05` |
| | Summary | `padding-right` | - | `$spacing-05` |

| | | `padding-right` | - | `$spacing-05` |
