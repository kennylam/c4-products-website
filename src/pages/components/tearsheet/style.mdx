### Typography

Tearsheet titles and labels should be set in sentence case. Keep all titles and labels concise and to the point. Tearsheet labels are optional.

| Class | Font-size (px/rem) | Font-weight | Type token |
| :-- | :-- | :-- | :-- |
| `.bx--modal-header__label` | 12 / 0.75 | Regular / 400 | `$label-01` |
| `.bx--modal-header__heading` | 20 / 1.25 | Regular / 400 | `$heading-03` |
| `.bx--modal-content` | 14 / 0.875 | Regular / 400 | `$body-long-01` |

### Zones

#### Header
![Wide tearsheet header](images/styles/wide/03.png)

#### Main content area
Table scrolling should keep a sticky header and roll the content underneath it.
![Wide tearsheet content scrolled to the top](images/styles/wide/mca-1.png)
![Wide tearsheet content scrolled down](images/styles/wide/mca-2.png)

Content that includes tables should expand to take advantage of the real estate but leave 16px of padding on both sides within the content area for type alignment. The cell of table rows should not be the same as the background color.
![Wide tearsheet table](images/styles/wide/mca-3.png)

Maintain 32px around the top, right, and left of the main content area.

#### Navigation buttons
Buttons scale to the content area of the *main content area* by percentage (being 25%) but have a max width of 312px.
![Wide tearsheet buttons](images/styles/wide/04.png)

| Breakpoint | Full-width | 256px influencer | 320px influencer |
|:--|:--|:--|:--|
| Max (1584px) | max width = 312px | max width = 312px | max width = 312px |
| X-Large (1312px) | 25% of content area = 312px | 25% of content area = 248px | 25% of content area = 232px |
| Large (1056px) | 25% of content area = 248px | 25% of content area = 184px | 25% of content area = 168px |
| Medium (672px) | - | - | - |
| Small (320px) | - | - | - |

#### Influencer _(Optional)_
Use 32px padding around the entire influencer to allow for alignment of the influencer content to the title of the tearsheet.

![Wide tearsheet influencer](images/styles/wide/inf-1.png)

#### Stacking
Stacking leaves the second layer 16px below the first layer but reduces the width of the first layer by 16px on each side.

![Wide tearsheet stacking](images/usage/stacking/stacking-02.png)
