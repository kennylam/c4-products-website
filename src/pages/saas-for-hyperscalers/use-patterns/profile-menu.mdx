<PageDescription>

The **Profile menu** is a persistent UI element that provides user, product and active session information. It is only available in the logged in state.

</PageDescription>

### Anatomy

The Profile menu describes the user, product instance, and MCSP and SaaS Console management resources:

![profile](images/userProfileMenu.png)

1. **User information:** User's name and email address.
2. **Instance information:** (Read only) Product instance name, region, and owner.
3. **User management:** (Optional) Access to MCSP user management if not available in-product.
4. **IBM SaaS Console:** UI for managing SaaS subscriptions.
5. **Logout button**: Log out of UI.
