### Dropdown tab order

The first tab stop is the search field. The second tab stop contains the Types the user is able to scope by. The user can navigate between the Types using the left and right arrow keys. Tapping 'Enter' will select and apply a scoping filter.
The third tab stop contains the recent searches. The user can navigate between the each of the options using the up and down arrow keys.
<Row>
<Column colMd={4} colLg={8}>

![Dropdown tab order](/images/DropdownTabOrder.png)

</Column>
</Row>


<br/>
<br/>

When the user selects a Type, the focus moves back up to the cursor in the search field. 
The backspace key will remove the Type tag. Tapping the Tab key will navigate the user to recent searches.

### Results tab order

The interactive components in each result tile are within the page's tab order.
To navigate between tiles' first tab stop (the result title), the user may use the up and down arrow keys.

### Filters tab order
The single select and multiselect filters follow the the standard keyboard annotation guidelines noted in Carbon. 
Single select filters utilize the [radio button](https://www.carbondesignsystem.com/components/radio-button/accessibility) component while multiselect filters utilize the 
[dropdown](https://www.carbondesignsystem.com/components/dropdown/accessibility) component. Both single and multiselect filters may include the [checkbox](https://www.carbondesignsystem.com/components/checkbox/accessibility) component.
