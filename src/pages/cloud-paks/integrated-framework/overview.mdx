## In-context enrichment

In-context enrichment is an important facet of integrated experiences as it provides the means for the user to access relevant information from across the platform when they need it, avoiding unneccesary navigation elsewhere. When relevant information is surfaced at the right time within a logical workflow, users feel as if the platform is working harder to keep them on track.

## Seamless navigation

Integrated, seamless navigation between applications and services is key to optimizing and maintaining fluid analysis within a workflow. Strategically placed navigation points within the platform experience promotes forward momentum and sets contextual pivot options for the user to navigate elsewhere across the platform.

Breadcrumb navigation across the platform reflects the hierarchical structure of an application/service. The returnTo banner supplements cross-platform navigation pivots with a simple back action as a means to seamlessly way-find back to that previous location on the platform.
