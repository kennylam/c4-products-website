#### Asset
- The Asset column provides a link to the asset specification, guidelines, or in-progress GitHub issue.
- Carbon community assets as tagged with `Carbon` and links to GitHue issues are tagged with `GHE issue`.
- Core assets that the majority of C&C products are expected to adopt are indicated with a green check mark (✅).

#### Maintainers
- Your point of contact for further information about the asset.

#### Design status
- `In progress`: a workgroup is currently working on the design specification.
- `Done`: the asset has been designed, evaluated, and approved by the DSAG / DSIT.
- When an asset has been approved and published on this site (or on the <a href="https://www.carbondesignsystem.com/community/overview/" target="_blank">Carbon community assets<Launch16 fill="#0F62FE" style="vertical-align:middle; margin: 0 2px 2px 0"/></a> page), 
code implementation can begin.

#### Code status
Once an asset is `Done` from a design perspective, work can begin on implementing it in code.
- `NA`: Code implementation work not yet possible as the design is still in progress.
- `Needs dev resource`: The design is done and we're now waiting for a development resource to get this asset implementd in code. If you think you can help, see our <Link to="/contributing/coding-components-patterns/overview">Coding components & patterns</Link> page.
- `In progress`: a developer is working on implementing the asset.
- When the code has been built, tested, and approved, a link is provided to our <a href="https://ibm-cloud-cognitive.netlify.app/?path=/story/getting-started--page" target="_blank">Storybook<Launch16 fill="#0F62FE" style="vertical-align:middle; margin: 0 2px 2px 0"/></a> showing the runing code.
- When complete, our coded assets are added to one of the following two React libraries: 
  - The <a href="https://github.com/carbon-design-system/ibm-cloud-cognitive" target="_blank">IBM Cloud & Cognitive React library<Launch16 fill="#0F62FE" style="vertical-align:middle; margin: 0 2px 2px 0"/></a>
  - The <a href="https://github.com/carbon-design-system/carbon-components-react" target="_blank">Carbon React library<Launch16 fill="#0F62FE" style="vertical-align:middle; margin: 0 2px 2px 0"/></a>

#### Further guidance
- For more information or updates regarding the table, please reach out to <Link to="https://w3.ibm.com/bluepages/profile.html?uid=2D1739897">Carrie Lloyd</Link>.
